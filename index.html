About Me

Building networks,
one connection at a time.

I'm a Jr. Network Specialist at the Lower Hudson Regional Information Center, supporting the Suffern, NY school district's network infrastructure day-to-day. My career path has taken me from consumer tech at Staples and Apple repair at Geek Squad to enterprise IT at Langan Engineering — and now into dedicated network work.

Outside of work, I run a personal home lab on repurposed hardware where I've built and operate a full self-hosted infrastructure stack: Proxmox virtualization, TrueNAS NFS storage, Docker containerization, Authentik SSO, Nginx Proxy Manager, Pi-hole DNS filtering, and Unifi networking — all running on my own subnet with proper UID/GID permission mapping and automated CI/CD via git polling.

I'm currently pursuing a B.S. in Cloud and Network Engineering at WGU, hold my CCNA, and I am driven to grow into a full Network Engineer role.

Auto-Deploy CI/CD via Git Polling
Built a lightweight CI/CD pipeline using a bash polling script that runs every 5 minutes via cron. Compares local HEAD against origin/main on GitHub — pulls and restarts Docker containers only on changes. Chose polling over webhooks deliberately to avoid exposing port 443 and minimize attack surface.
